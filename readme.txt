This app lets you track your expenes by having a .py script expense_tracker with a class of all core functionality.
And the script tracker.py to actually use the thing. 
It accept's user input to execute functions like add, expense and status, after you are done running the script you should close it with close function, creating a data.pickle file, which stores previous data and provides it upon executing the code again.

To run the program simply run the tracker.py script.
To delete the old data, delete data.pickle file
